프로젝트 소개<br>
사회적으로 유기동물 문제에 대한 인식의 부족을 인지하게 되었습니다. 
이는 많은 유기동물들이 새로운 가정을 찾지 못하고
보호소에서 일상을 보내야 하는 현실로 이어집니다. 
우리는 이러한 문제를 해결하고자, 유기 동물들의 정보를 제공하여
더 많은 동물이 사랑받고 편안한 삶을 살 수 있게 하는 것을 목표로 설정했습니다.
